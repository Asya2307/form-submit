Задача

При нажатии на button class="submit" получить значения из всех полей и провалидировать их - ни одно из полей не должно быть пустым, валидным полям добавлять .is-valid, инвалидным .is-invalid.

Если все поля валидные, то вывести полученные значения в соответствующие блоки:

  • Значение из input class="email" в p class="email-output"
  • Значение из input class="password" в p class="password-output"
  • Значение из input class="confirm-password" в p class="confirm-password-output"

Пошаговый план выполнения

  1. Получаешь form class="form" и вешаешь на него событие submit
  2. В коллбеке получаешь event и делаешь event.preventDefault()
  3. Получаешь поля и сохраняешь их в переменные
  4. Получаешь значения полей и сохраняешь их в переменные
  5. Создаешь переменную и сохраняешь туда массив объектов по типу
    [{ element: field1, value: value1 },
    { element: field2, value: value2 },
    { element: field3, value: value3 }]
  6. Начинаешь переберать этот массив с помощью .forEach, внутри массива проверяешь value с помощью .length и если оно равно 0, то добавляешь этому полю .is-invalid и удаляешь .is-valid, а если length больше 0, то добавляешь .is-valid и удаляешь .is-invalid
  7. Получаешь input class="is-invalid" с помощью .querySelectorAll и сохраняешь в переменную
  8. Получаешь .length этого массива и если он равен 0, то выводишь полученные значения полей в соответствующие блоки
